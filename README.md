Payroll Management System (Flask Web Application)
A simple and efficient Payroll Management System built using Flask, designed to help manage employees, record attendance, and generate monthly salary automatically. This project is created as part of the ICT Academy Full Stack program and demonstrates core concepts like authentication, CRUD operations, database handling, and deployment.
Features
User Login
Admin login to securely access the system.
Employee Management
Add employees with name, designation, and basic pay.
View all employees in a clean dashboard.
Attendance Management
Record attendance as number of days present.
Update attendance data anytime.
Payroll Processing
Automatically calculates monthly salary using:
Salary = (Basic Pay / 26) × Present Days
Generates downloadable PDF payslips.
Database
Uses SQLite (payroll.db) managed via SQLAlchemy ORM.
Deployment
Fully deployed on PythonAnywhere.
Accessible online via hosted link.
Technology Stack
Backend: Python, Flask
Frontend: HTML, CSS
Database: SQLite (SQLAlchemy)
PDF Generation: ReportLab
Hosting: PythonAnywhere
